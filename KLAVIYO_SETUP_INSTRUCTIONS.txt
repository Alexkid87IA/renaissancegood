===============================================================
ETAPE 1 : CONNECTER SHOPIFY A KLAVIYO
===============================================================

1. Aller dans Klaviyo > Settings (engrenage) > Integrations
2. Cliquer "Add Integration" > chercher "Shopify"
3. Entrer l'URL de la boutique Shopify Renaissance
4. Autoriser la connexion
5. Activer la synchronisation :
   - Placed Order (commandes)
   - Started Checkout (panier abandonne)
   - Viewed Product (navigation)
   - Customer data (profils)
6. Sauvegarder

Sans cette etape, les flows abandoned cart et post-purchase
ne fonctionneront pas.

===============================================================
ETAPE 2 : FLOW "WELCOME EMAIL" (bienvenue newsletter)
===============================================================

1. Aller dans Flows > Create Flow > "Build your own"
2. Nom : "Welcome Series - Renaissance"
3. Trigger : "When someone is added to a list" > "Newsletter" (YA9SMX)

4. AJOUTER UN CONDITIONAL SPLIT (pour la langue) :
   - Condition : Profile property "language" equals "en"
   - Branche YES (anglais) :
     -> Email avec template "Welcome Email EN" (VNUqTu)
        Subject : "Welcome to the Maison Renaissance"
        From : Renaissance Paris <contact@renaissance-eyewear.fr>
   - Branche NO (francais, par defaut) :
     -> Email avec template "Welcome Email FR" (QUyiDG)
        Subject : "Bienvenue dans la Maison Renaissance"
        From : Renaissance Paris <contact@renaissance-eyewear.fr>

5. Passer le flow en "Live"

COMMENT CA MARCHE : Le site envoie automatiquement la propriete
"language" (fr ou en) a Klaviyo quand quelqu'un s'inscrit via
le footer. Le conditional split route vers le bon template.
